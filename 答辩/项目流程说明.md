# 中国东北-俄罗斯远东跨境区域SDGs空间支撑条件评估与发展对策 — 项目流程说明

## 研究主线

```text
2010/2015/2020/2024 四期年度或准年度空间数据
→ 10km 网格指标计算
→ 全时期合并极差标准化
→ 逐年维度内熵权法与维度间等权
→ TOPSIS 相对贴近度 SDG 综合指数
→ 综合指数时空格局与变化率分析
→ ESDA 空间集聚特征识别（Global Moran's I + LISA）
→ 维度组合分析与短板识别
→ 省州/国别分组统计与跨境差异解释
→ QGIS 正式制图与论文写作
```

论文表述为"2010年以来至近年"，不把 2025 写成完整模型年份。当前模型最新完整年份为 2024；2025 只作为后续数据更新方向。

## 投影规范

| 用途 | 投影 | 说明 |
|------|------|------|
| 原始数据存储、GEE 裁剪边界、Web 预览 | `EPSG:4326` | WGS84 |
| 面积、距离、网格尺度、口岸距离、QGIS 正式制图 | `ESRI:102025` | Asia North Albers Equal Area Conic |

不使用 `EPSG:3857` 做面积、距离或网格尺度分析。

## 数据处理流程

### 步骤 1：环境准备

```bash
source .venv312/bin/activate
```

使用 `.venv312/bin/python`，不依赖系统 `python`。

### 步骤 2：生成 10km 网格

**脚本**：`scripts/generate_10km_grid.py`

**输入**：`data/study_area_no_holes.gpkg`（去洞研究区边界）

**输出**：`data/study_area_grid_10km.gpkg`（10km 网格，ESRI:102025 投影，仅保留质心落在去洞研究区内的网格）

**配置**：`scripts/workflow_config.py` → `TARGET_RESOLUTION = 10000`, `METRIC_CRS = "ESRI:102025"`

```bash
python scripts/generate_10km_grid.py
```

### 步骤 3：GEE 数据下载

**脚本**：`scripts/gee_download_multiyear.py`

**输入**：GEE 在线数据集（GPW、VIIRS、MODIS、TerraClimate、GTOPO30）

**输出**：`data/gee_multiyear/*.tif`（各指标四期 GeoTIFF 栅格）

**配置**：`scripts/workflow_config.py` → `TARGET_YEARS`, `YEAR_MAPPING`, `INDICATORS`

```bash
export EARTHENGINE_PROJECT=sdgs-china-russia-border
FORCE_GEE_EXPORT=1 python scripts/gee_download_multiyear.py
python scripts/check_gee_tasks.py
```

GEE 导出完成并将 GeoTIFF 放入 `data/gee_multiyear/` 后继续下一步。

### 步骤 4：通达性距离计算

**脚本**：`scripts/pair_border_ports.py` → `scripts/calculate_multiyear_indicators.py`

**输入**：口岸矢量数据、城市点数据、道路数据、10km 网格

**输出**：
- `dist_port`：网格质心至最近口岸距离（km，ESRI:102025 投影下计算）
- `dist_city`：网格质心至最近城市/城镇节点距离（km）
- `dist_major_road`：网格质心至最近主要道路距离（km）

> 对应终稿D2通达性维度的三个指标。

### 步骤 5：网格指标计算

**脚本**：`scripts/calculate_multiyear_indicators.py`

**输入**：`data/gee_multiyear/*.tif` + `data/study_area_grid_10km.gpkg`

**输出**：`data/grid_indicators_{year}.gpkg`（2010/2015/2020/2024 四期指标）

**方法**：栅格质心采样 → 网格属性表

```bash
python scripts/calculate_multiyear_indicators.py
```

### 步骤 6：SDG 综合指数计算（对应终稿3.3节）

**脚本**：`scripts/calculate_multiyear_index.py`

**输入**：四期 `grid_indicators_{year}.gpkg`

**输出**：
- `data/grid_sdg_multiyear.gpkg`（多年 SDG 综合指数 + 四维得分）
- `data/weights_multiyear.csv`（熵权权重）
- `data/fixed_weights_multiyear.csv`（固定权重副本）
- `data/normalization_ranges.csv`（极差标准化范围）
- `data/outlier_clip_ranges.csv`（异常值裁剪范围）

**方法链**（与终稿3.3节一致）：
1. 1%和99%分位数极值剪裁，降低异常值影响
2. 全时期合并极差标准化（正向：`(x-min)/(max-min)`，负向：`(max-x)/(max-min)`）
3. 维度内熵权法（四期标准化样本合并，按信息离散度分配指标权重）
4. 维度间等权（各0.25，避免人口密度、夜间灯光等高离散指标过度主导）
5. TOPSIS 相对贴近度（正理想解=1，负理想解=0）

```bash
python scripts/calculate_multiyear_index.py
```

### 步骤 7：综合指数时空格局与变化率分析（对应终稿4.1节）

**脚本**：`scripts/analyze_trends_and_hotspots.py`

**输入**：`data/grid_sdg_multiyear.gpkg`

**输出**：
- `data/trend_analysis.gpkg`（变化率、年均变化率）
- `data/trend_statistics.csv`
- `data/hotspot_statistics.csv`
- `data/temporal_correlation.csv`

**方法**（与终稿4.1节对应）：
- 四期综合指数描述统计（均值、中位数、箱线图）
- 阶段变化率：`(2024值 - 2010值) / 2010值`
- 年均变化率（AAGR）：`(2024值/2010值)^(1/14) - 1`
- 省州级综合指数排名与统计

```bash
python scripts/analyze_trends_and_hotspots.py
```

### 步骤 8：ESDA 空间集聚特征分析（对应终稿3.4节和4.2节）

**脚本**：`scripts/analyze_esda_spatial_clustering.py`

**输入**：`data/grid_sdg_multiyear.gpkg`

**输出**：
- `data/esda_lisa_clusters.gpkg`（LISA 集聚类型：高-高/低-低/高-低/低-高）
- `data/esda_global_moran.csv`（全局 Moran's I 统计量）
- `data/esda_lisa_statistics.csv`（LISA 局部统计量）

**方法**（与终稿3.4节和4.2节一致）：
- 空间权重矩阵：Queen 一阶邻接（共享边界或顶点）
- 全局 Moran's I：检验整体空间自相关程度
- 局部 LISA：识别 HH/LL/HL/LH 四类集聚类型
- 分别对四期综合指数和2010-2024年变化率计算 Global Moran's I 和 LISA

> 终稿4.2节结果：四期综合指数 Global Moran's I 分别为 0.8960、0.9013、0.9125、0.9179（p=0.001），空间集聚程度逐步增强。变化率 Moran's I 为 0.5237。

```bash
python scripts/analyze_esda_spatial_clustering.py
```

### 步骤 9：维度组合分析与短板识别（对应终稿4.3节）

**脚本**：`scripts/analyze_lisa_dimension_shortboards.py`

**输入**：`data/esda_lisa_clusters.gpkg` + `data/grid_sdg_multiyear.gpkg`

**输出**：
- `data/lisa_dimension_shortboard_statistics.csv`（各集聚类型的维度短板统计）
- `data/lisa_indicator_shortboard_details.csv`（指标层面短板详情）

**方法**（与终稿4.3节一致）：
- 比较D1人类活动、D2通达性、D3生态状态、D4环境本底四个维度得分
- 中俄两侧维度得分对比
- LISA类型区维度均值统计与热力图
- 识别各集聚类型的主短板和次短板维度及指标

> 终稿4.3节结果：两侧最强维度均为D2通达性，短板维度均为D1人类活动。HH与LL集聚区差值最大维度为D3生态状态（0.5906），其次为D2通达性（0.3650）。

```bash
python scripts/analyze_lisa_dimension_shortboards.py
```

### 步骤 10：行政分组与省州统计（支撑终稿4.1和4.3节）

**脚本**：`scripts/add_admin_grouping.py`

**输入**：`data/grid_sdg_multiyear.gpkg` + GADM 4.1 ADM1 行政边界

**输出**：
- `data/grid_admin_grouped.gpkg`（带国别/省州分组的网格）
- `data/admin1_group_statistics.csv`（省州级统计）
- `data/admin1_hotspot_statistics.csv`（省州级热点统计）
- `data/country_group_statistics.csv`（国别级统计）

**方法**：10km 网格质心匹配 GADM ADM1 行政区划 → 按国别/省州分组统计

```bash
python scripts/add_admin_grouping.py
```

### 步骤 11：质量报告

**脚本**：`scripts/generate_quality_report.py`

**输入**：所有中间输出

**输出**：数据质量报告（含年份映射说明、缺失值统计、权重诊断）

```bash
python scripts/generate_quality_report.py
```

## 注意事项

1. **2010年夜光数据**：使用2012年VIIRS数据作为起始期近似替代，论文中已披露，不能解释为2010年实测值。
2. **2024年人口数据**：使用2020人口基线，论文方法中已说明，不能解释为2024年人口观测值。
3. **投影转换**：涉及面积、距离计算时，务必使用 ESRI:102025 投影。
4. **ESDA方法**：空间权重矩阵采用Queen一阶邻接，LISA识别HH/LL/HL/LH四类集聚类型。
5. **行政分组**：优先使用 GADM 4.1 ADM1，按 10km 网格质心匹配。
6. **赋权方法**：维度内熵权法（按数据离散信息量分配）+ 维度间等权（各0.25，保证公平）。
7. **维度名称**：D1 人类活动、D2 通达性、D3 生态状态、D4 环境本底（与终稿表3-1一致）。
8. **论文结构**：第4章结果包含4.1综合指数时空格局、4.2 ESDA空间集聚特征、4.3维度组合分析；第5章讨论包含网格尺度意义、短板结构、中俄互补、外部性与制度边界、协同治理路径。
